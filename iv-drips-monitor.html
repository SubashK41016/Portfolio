IV DRIPS MONITOR

IoT & Healthcare Project

A Smart IoT-based healthcare solution designed to monitor intravenous (IV) fluid levels and drip rates in real-time, reducing nurse workload and improving patient safety.

Technologies Used

React 19 Vite 7 Firebase Auth ESP8266 Load Cell (HX711) Capacitor (Android) Recharts

Problem Statement

In hospitals, nurses must manually monitor IV bottles across multiple patients. If IV fluid finishes unnoticed, air can enter the bloodstream (air embolism). Drip rate inconsistencies may go undetected, affecting medication dosage. Manual tracking increases workload, stress, and risk of human error. There is a need for an automated, real-time IV monitoring system.

Key Features

Real-Time Monitoring

Continuously tracks fluid levels and drip rate. Data updates dynamically with animated progress bars.

Automatic Alerts

Generates instant color-coded warnings (Yellow) or critical alerts (Red) when fluid is low or flow stops.

Multi-Patient Dashboard

Centralized view to track multiple patients simultaneously. Add or edit patient details dynamically.

Live Trend Charts

Interactive data visualization using Recharts to analyze historical fluid drainage and rate fluctuations.

Secure Access

Protected staff access backed by Firebase Authentication with session permanence.

Cross-Platform

Fully responsive web application and an installable native Android App built via Capacitor.

System Architecture

The system comprises an IoT hardware layer that captures weight data and transmits it to the cloud. The frontend React application fetches this real-time data to render a unified dashboard.

Impact & Results

  • Prevents IV Empty Risk: Eliminates the severe danger of air embolism by notifying staff before the bottle drains.
  • Instant Triage: A traffic-light color system (Green, Yellow, Red) ensures medical personnel can identify critical patients at a glance.
  • Reduced Workload: Automates the manual checking process, allowing nurses to focus on direct patient care rather than constant supervision.
  • Cost-Effective: Uses accessible IoT components (ESP8266) coupled with free-tier cloud services for a highly scalable solution.
