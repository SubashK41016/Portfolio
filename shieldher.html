ShieldHer Safety Band

IoT & Mobile Project

An IoT-powered wearable safety device with real-time GPS tracking, instant SOS alerts, and a specialized police patrol command center to ensure women's safety.

Technologies Used

ESP32 Microcontroller Node.js & Express Leaflet.js NEO-6M GPS & SIM800L MPU6050 Accelerometer Firebase Auth

Key Features

Instant SOS Alerts

A 3-second hold on the physical band or companion app button triggers an instant emergency alert, notifying contacts and the police dashboard immediately.

Real-Time GPS Tracking

Live location tracking backed by a robust Leaflet.js map with satellite and street map toggle functionality.

Police Command Center

A specialized dashboard for patrol authorities providing a live alert feed, incident mapping, and immediate respond/resolve management tools.

Danger Zones & Heatmaps

Visual heatmaps featuring risk-level indicators based on reported incidents to identify top zones and historical risks.

Fall Detection Integration

Built-in MPU6050 accelerometer detects potential falls and automatically triggers an SOS post a 10s wait buffer, mitigating false positives.

GSM SMS & Offline Modes

Even without Wi-Fi, the integrated SIM800L module sends backup emergency SMS messages with Google Maps location data to trusted contacts.

System Architecture

ShieldHer bridges physical IoT hardware with cloud-based visualization:

  • Hardware (ESP32): Processes sensor data (pulse, GPS, accelerometer) and transmits payloads via Wi-Fi or GSM. Features haptic feedback (vibration) and buzzer alerts.
  • Backend (Node.js): REST API managing live incidents, active contacts, Danger Zone statistics, and real-time ESP32 heartbeat pulses.
  • Frontend Web App: A Progressive Web App (SPA shell) tailored for mobile users with easy SOS triggering, secure Firebase authentication, and interactive layers.
  • Police Dashboard: Desktop-optimized interface for authorities that continuously polls for live updates and allows immediate dispatch assignment.
